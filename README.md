# Smart Queue Mobile App

## Overview

Smart Queue is a mobile application designed in Figma to reduce waiting times by allowing users to join virtual queues, book appointments, and receive live queue updates.

## Problem

Long waiting lines at hospitals, banks, and government offices waste people's time.

## Solution

The Smart Queue app allows users to:

- Join virtual queues
- Book appointments
- Track waiting time
- Receive live notifications
- Scan QR codes for check-in
- Navigate to service branches

## Tools Used

- Figma

## Features

- Splash Screen
- Login Screen
- Home Dashboard
- Live Queue Tracking
- QR Check-in
- Notifications
- Profile
- Settings
